问题1：我想给所有的文件名开头加一个前缀“PRE_”应该怎么做？

回答1：for f in * ; do mv -- "$f" "PRE_$f" ; done

问题2：我想给所有的文件名末尾加一个后缀“.pdf”应该怎么做？

回答2：for f in * ; do mv -- "$f" "${f}.pdf" ; done

问题3：我想把所有的文件名末尾“.prog”替换成“.prg”应该怎么做？

回答3：for f in *.prog; do mv -- "$f" "${f%.prog}.prg" ; done

问题4：我想把所有的文件名开头“xha”替换成“xhd”应该怎么做？

回答4：for f in xha*; do mv -- "$f" "xhd${f#xha}" ; done

问题5：我想把所有的.mp3文件名内部第1个“xha”替换成“xhd”应该怎么做？

回答5：for f in *.mp3; do mv -v "$f" "${f/xha/xhd}"; done;

问题1：我有一堆文件的filename包含orbit轨道号，如果我想找到orbit_48912～orbit_48923的文件，怎么搜索呢？ls *orbit_489[1-2][2-3]*只能找到4个文件，正确的命令应该怎么写？

回答1：用大括号匹配ls *orbit_489{12..23}*

问题2：承接问题1，ls *orbit_489[1-2][2-3]*类似可以通过find . -name "*orbit_489[1-2][2-3]*"实现，ls *orbit_489{12..23}*如何用find同理实现？

回答2：未解决。应该用正则表达式。

问题3：如何显示本路径下的文件（或文件夹）数目？

回答3：统计当前目录下文件的个数（不包括目录）$ ls -l | grep "^-" | wc -l（详细和拓展，参考SnailTyan）

问题4：如何linux ls命令目录显示前10个文件（alphabetically排序）？

回答4：ls -l | head -n 10 但如果需要按照生成时间排序（越早越靠前）就要把-l改为-ltr

问题5：如何查看一个文件夹的大小／尺寸／容量？

回答5: du -sh（参考linuxize.com）应当注意，如果一个文件夹特别巨大，显示容量的反馈时间会特别长，以至于无法继续工作，所以不知道多大的时候建议另开screen进行查看操作。
